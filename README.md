# 🔍 Digital Evidence

Repositori ini berisi proyek halaman web statis yang dirancang untuk mendokumentasikan dan menampilkan temuan bukti digital (*digital evidence*). Dibangun menggunakan HTML dan CSS murni, antarmuka web ini menyajikan aset-aset gambar bukti secara terstruktur dan mudah diakses.

## 📂 Struktur Repositori

Proyek ini terdiri dari komponen web dasar dan kumpulan aset gambar. Berikut adalah rincian file di dalamnya:

- `index.html` — File utama yang memuat kerangka dasar dan struktur konten halaman web.
- `style.css` — File stylesheet untuk mengatur tata letak, warna, dan antarmuka (UI) agar halaman tampil rapi.
- `1.png`, `1A.png`, `2.png`, `3.png` — Kumpulan aset gambar yang merepresentasikan dokumentasi bukti digital utama.
- `bonus.png` — Aset gambar tambahan.
- `README.md` — File dokumentasi repositori.
